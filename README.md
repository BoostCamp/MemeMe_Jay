## MemeMe App 2.0

#### 기본기능

* Meme 이미지 만들기
* Table View, Collection View로 출력하기

#### 추가기능

1. 수정기능
  * 글, 사진 수정가능.
  * 수정된 것 공유가능.

2. 테이블 뷰에서 삭제 가능
  * 왼쪽으로 드래그 시 Delete가능.
